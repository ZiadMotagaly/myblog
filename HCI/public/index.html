Lab1

HCI (PC) In this Lab, I designed a game called Roll- A -Maze , where the user neads to control the ball using Keyboard Inputs , to navigate through the maze and collect points to win the game. The arrow keys move the ball in their respective directions, and the space key makes the ball jump. The User needs to collect the green cubes to increment the counter by 1.

Lab2

HCI (Android) In this Lab, I deployed the Roll- A -Maze game that we created in Lab 1 , on an Android smartphone. This is done through Unity as well, by modifying the Build and Player settings. The main difference is that our input will now be set to the controls embedded in modern smart phones. I implemented both a Gyroscope and an Acceloremeter into the Input settings of the ball.

Posted Jan 29, 2023

Lab3

HCI (Oculus VR Headset) For Lab 3 , I deployed the Roll- A -Maze game that we created in Lab 1 , on an Oculus headset. In this case, we designed our own XR Hand Controller to the replace the standard Input. A virtual reality headset is a head-mounted device that provides virtual reality for the wearer. These headsets include a stereoscopic display (providing separate images for each eye), stereo sound, and sensors like accelerometers and gyroscopes for tracking the pose of the user's head to match the orientation of the virtual camera with the user's eye positions in the real world.
